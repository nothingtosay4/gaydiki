Гайд: Активация Windows 11/10 бесплатно

Пошаговая активация Windows PowerShell

  1. Cmd (Win+R) для PowerShell
    Нажмите Win+R, введите powershell и запустите от администратора для активации Windows
  2. KMS команда активации
    Вводим команду KMS активации: irm get.activated.win | iex - убирает надпись активация Windows
  3. Готово! Windows активирована
    Надпись исчезла навсегда. Проверка: Настройки → Активация
